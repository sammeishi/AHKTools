# AHKTools
base autohotkey tools



- **BluetoothSpeaker996**: Every once in a while, a sound that is inaudible to human ears occurs, keeping the Bluetooth speaker connected and preventing the Bluetooth speaker from disconnecting.
- **HotkeyCountdownTimer**: Shortcut keys activate the countdown timer on the screen, full keyboard control, simple and beautiful
- **SamHotKeys**: Sam's customized shortcut keys
